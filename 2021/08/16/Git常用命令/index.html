Git常用命令

2021年11月5日 下午

一些平时经常使用的命令

查看状态

git status

初始化本地库

git init

在本地磁盘下创建一个文件夹，使用git进入该文件夹，使用git init进行初始化，初始化后不要修改生成的文件

将文件添加到暂存区

git add 文件名

从暂存区删除文件

git rm –cached 文件名

将暂存区文件提交到本地库

git commit -m “说明信息” 文件名

查看版本日志信息

git reflog,简略

git log,完整信息

版本穿梭(切换版本)

git reset –hard 版本号

查看分支

git branch -v

创建分支

git branch 分支名

切换分支

git checkout 分支名

切换回master git checkout master

合并分支

git merge 分支名

冲突合并

两个分支在同一个文件的同一个位置，有两套不同的修改，git无法决定使用哪一个，必须人为决定

进入文件，将文件中的《《《《 和====和》》》内容删除，保留想要更改的内容

然后提交暂存区、提交本地库。提交本地库时，不加文件名，例如 git commit -m “test”

查看远程库别名

git remote -v

创建远程库别名

git remote add 别名 链接

别名最好和仓库名相同

推送远程库

git push 别名(或者链接名) 分支名

拉取远程库

git pull 别名(或链接名) 分支名

克隆远程库到本地

git clone 链接

生成ssh免密登录

ssh-keygen -t rsa -C
