Vue cli

今天是农历大年三十，现在是凌晨12点21，刚才成功启动了一个自己创建的vue类的项目，有点兴奋，但也有点发愁

兴奋是因为解决了自己终于知道怎么使用前端的东西，而不是像两天前，问那种傻×的问题，两天能找到方法，觉得已经挺欣慰的。现在，i know，可以使用vue cli快速创建项目，就能生成那些可以交互的东西了，应该是这样，还没深入去用.

发愁是因为跟着做的那个项目还没做完，还剩四天左右就可以做完，但和老师说的是15天搞定毕设的后端界面，那只有使用现成的vue-admin去改了。前端现在也有点不太懂，懵的状态，慢慢学吧，了解的越多，越清晰，也会觉得自己越垃圾，也会有一点眼高手低。
